# Traffic Detection System 🚦

This project is a **Traffic Detection System using YOLOv8 and Django** that detects vehicles from video or images using computer vision.

## Features

* Vehicle detection using YOLOv8
* Django web interface
* Traffic monitoring
* Real-time object detection

## Technologies Used

* Python
* Django
* YOLOv8
* OpenCV
* HTML/CSS/JavaScript

## Installation

1. Clone the repository

```
git clone https://github.com/AjayBabu-1423/Traffic-detection-system.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the Django server

```
python manage.py runserver
```

4. Open browser

```
http://127.0.0.1:8000
```

## Project Structure

```
APP/        -> Django app files
PROJECT/    -> Django settings
model/      -> YOLO model
static/     -> CSS and JS files
```
